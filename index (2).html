DevOps Life Cycle for Quiz Game

1. Planning

Define quiz game goals & features.

2. Development

Write HTML, CSS, and JavaScript code.

3. Build

Organize files for deployment.

4. Testing

Test game manually & in multiple browsers.

5. Deployment

Upload to GitHub Pages for public access.

6. Operation

Users play the game online.

7. Monitoring

Collect feedback & fix issues for updates.
